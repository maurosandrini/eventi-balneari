# Eventi in spiaggia — Punta Marina & Marina di Ravenna

Calendario degli eventi serali (concerti, serate, presentazioni) degli stabilimenti balneari di Punta Marina e Marina di Ravenna.

App statica: `index.html` legge `events.json`. Dati raccolti da fonti pubbliche (ticketing, stampa locale, agenda del Comune), aggiornati periodicamente. Utility privata, non commerciale. Verifica sempre data e orario sulla pagina del locale.

Legenda stelle: 3 = culturale, 2 = concerti/musica, 1 = culinario.
